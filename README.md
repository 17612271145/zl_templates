# h5

开发阶段入参

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### 构建
```
npm run build
```


### Technical route
vue2.0 + mock + vue-router + axios + vuex + typescript + vant + eslint + sass + vuecli3
```

### 目录结构
    api:接口请求地址
    assets:存放静态资源图片地址
    components:存放公共组件地址
    layout:全局布局文件
    router:路由配置文件
    store:状态仓库
    views:核心代码存放地址
